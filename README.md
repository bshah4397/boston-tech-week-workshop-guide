# Boston Tech Week Embedded Apps Workshop Guide

Participant-facing guide site for the Embedded App Framework workshop.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Purpose

This repo is independent from the workshop app-host repo. It contains the guide,
prompts, troubleshooting notes, and live workshop path. It does not contain
SMART OAuth client IDs, app registrations, access tokens, or Athena credentials.
